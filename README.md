# Py-Projects
A collection of logic tests and simulations I've created using P3.

App Descriptions:

Car Shop - A simple car inventory simulation game. You start out with a collection of three cars and given the option to either add or change them. Use 'quit' at any time to end the simulation.

Mind Reader - Another simple game where you guess what number the bot is thinking of. I've utilized randint and seed to generate a random number which you are then given a prompt to guess. The code works so that each time you play a new random number is generated for every loop.

Prompt Calculator - A calculator app that performs basic math equations. It first asks you what type of function you would like to perform then asks for the two numbers to perform the calculation on. I've also added an exception for dividing by zero to avoid an error.

Rock Paper Scissors - A game of rock paper scissors. You tell the game which hand you choose then it utilizes randint to generate a random number which is then assigned to a variable. From there it goes through a set of if statements to determine the outcome of the game based on each hand played. I've also incorporated a countdown feature to build suspense and score counter which indicates wins vs losses.
